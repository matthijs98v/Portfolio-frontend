# Portfolio Website

This is my portfolio website, made with vue, vue-router and tailwindCss and github actions for auto deployment. It also has a backend [api](https://github.com/matthijs98v/Portfolio-backend)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
